# 🧠 Alzheimer’s Prediction, Progression Analysis, and Patient Support System

## **Overview**
This project is a Machine Learning-based system that helps predict Alzheimer's disease risk, analyze cognitive decline progression, and provide support for patients and caregivers. The system uses cognitive assessments such as MMSE and MoCA along with user health and lifestyle data to generate predictions and recommendations.

## **Features**
- User Registration & Login
- MMSE and MoCA Cognitive Assessments
- Alzheimer's Risk Prediction
- Cognitive Stage Classification
- Progression Analysis
- Personalized Health Recommendations
- Caregiver Alert System
- Interactive Dashboard

## **Technologies Used**
- **Frontend:** HTML, CSS, JavaScript
- **Backend:** Python, Flask
- **Machine Learning:** Scikit-learn, Pandas, NumPy
- **Database:** SQLite/MySQL

## **Machine Learning Models**
- **Random Forest** – Cognitive stage classification and risk prediction
- **Linear Regression** – Progression forecasting and trend analysis

## **Project Workflow**
1. Collect user information and assessment scores.
2. Preprocess and analyze the data.
3. Predict Alzheimer's risk and cognitive stage.
4. Forecast future cognitive changes.
5. Generate recommendations and caregiver alerts.
