# Memora — Intelligent Memory Assistant

Memora is an intelligent memory-assist web application built with React and Vite, transforming daily care routines into actionable, data-driven insights through a polished, accessible interface designed for real-world caregiver and user impact.

## Why it matters
- Solves real-world usability and accessibility problems for memory-impaired users and their caregivers.
- Demonstrates polished front-end engineering, intuitive UX, and modular architecture suitable for portfolio presentation.

## Key features
- Clean, responsive UI with reusable components and theming support.
- Local sample database with utilities for data access and testing (`src/data`).
- Analysis dashboard for visualizing activity trends and caregiver panels for supervision.
- Modular component structure enabling easy extension and testing.

## Role and contributions
- Designed and implemented the front-end using React, Vite, and modern CSS modules.
- Built reusable UI components and responsive page layouts for a smooth user experience.
- Developed data utilities and a local mock database to simulate real user interactions.
- Created an analytics dashboard to surface actionable insights for caregivers.

## Technologies
- Frontend: React, Vite, JSX, CSS modules
- Tooling: ESLint, Vite dev server, npm
- Data: Local JSON mock database and modular test scripts in `src/data`

## Architecture overview
- `src/components` — Reusable UI components (buttons, cards, navbar)
- `src/pages` — Page-level views (Landing, Profile, Dashboard, Caretaker panel)
- `src/data` — Local mock database and helper utilities to access test data

## Quick start
1. Install dependencies

```bash
npm install
```

2. Run the development server

```bash
npm run dev
```

3. Build for production

```bash
npm run build
```

## Deployment
- The project builds to a static bundle that can be deployed to GitHub Pages, Netlify, or Vercel.
